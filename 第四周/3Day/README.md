## 一.通过Vue-cli创建项目

### 1.安装vue-cli
    
    cnpm install vue-cli -g

### 2.查看安装的vue-cli的版本

    vue -V
    
### 3.查看vue-cli提供的模板

    vue list
    
### 4.创建项目

    vue init 模板名称  项目名称
    
### 5.进入创建项目的目录

    cd 项目名称
    
### 6.安装项目所需要的依赖

    cnpm install
    
### 7.启动项目

    npm run dev
    
### 8.打包项目

    npm run build

## 三.组件之间的通信

### 1.父子组件之间的通信

#### 1.1 父传子
    
    父组件传递: <子组件名称 :属性名="要传递的数据"></子组件名称>
    子组件接收: props: []  / {}
    
#### 1.2 子传父

    子组件传递: this.$emit("监听的事件","传递的值")
    父组件接收:　<子组件名称 @监听的事件="接收的方法"></子组件名称>    

### 2.非父子组件的通信

    var bus = new Vue()   //中央事件总线
    
    传递: 
        bus.$emit("方法名","传递的值")
        
    接收:
        bus.$on("方法名",接收传递的值的回调函数)
            

### 3.跨级组件之间的通信 

 var bus = new Vue()
    
    传递: 
        bus.$emit("方法名","传递的值")
        
    接收:
        bus.$on("方法名",接收传递的值的回调函数)
